# ml-learning
 The project is based on SVM.
 The project detects whetehr an e-mail recieved by the user is a SPAM or HAM.
 
 Flowchart:
 1. Data(spam/ham mail)
 2. Data pre processing
 3. Feature extraction
 4. Train - Test - Split
 5. SVM model
 6. Trained SVM model
 7. Detect if mail is SPAM / HAM
